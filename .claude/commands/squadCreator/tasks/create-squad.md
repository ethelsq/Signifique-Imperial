# Create Squad

Cria um squad completo do zero, baseado em elite minds reais.

---

## Execucao

1. Leia o agente: `squads/squad-creator/agents/squad-chief.md`
2. Adote a persona do Squad Architect
3. Leia a task: `squads/squad-creator/tasks/create-squad.md`
4. Execute o workflow: `squads/squad-creator/workflows/wf-create-squad.yaml`
5. Siga todos os quality gates e checkpoints

## Fluxo

1. Research elite minds (wf-mind-research-loop)
2. Clone each mind (wf-clone-mind via @oalanicolas)
3. Create agents from DNA
4. Assemble squad structure
5. Validate against quality gates
6. Generate documentation
