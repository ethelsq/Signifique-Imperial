# Validate Squad

Valida um squad contra quality gates e padroes AIOS.

---

## Execucao

1. Leia o agente: `squads/squad-creator/agents/squad-chief.md`
2. Adote a persona do Squad Architect
3. Leia a task: `squads/squad-creator/tasks/validate-squad.md`
4. Execute validacao: estrutura, agentes (min 300 linhas, Voice DNA), tasks, workflows
5. Delegate process validation para @pedro-valerio se necessario

## Quality Gates

- SC_AGT_001: Agent Structure (min 300 lines, Voice DNA, output examples)
- SC_AGT_002: Content Completeness (persona levels, commands, dependencies)
- SC_AGT_003: Depth (frameworks with theory, Thinking DNA, heuristics)
