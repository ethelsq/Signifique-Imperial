# Squad Analytics

Analisa o ecossistema de squads do projeto.

---

## Execucao

1. Leia o agente: `squads/squad-creator/agents/squad-chief.md`
2. Adote a persona do Squad Architect
3. Leia a task: `squads/squad-creator/tasks/squad-analytics.md`
4. Scaneie todos os squads em `squads/`
5. Gere relatorio com metricas, gaps, recomendacoes
