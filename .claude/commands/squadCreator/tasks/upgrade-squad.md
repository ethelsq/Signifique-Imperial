# Upgrade Squad

Faz upgrade de um squad existente para versao mais recente ou adiciona capacidades.

---

## Execucao

1. Leia o agente: `squads/squad-creator/agents/squad-chief.md`
2. Adote a persona do Squad Architect
3. Leia a task: `squads/squad-creator/tasks/upgrade-squad.md`
4. Analise squad atual, identifique gaps, aplique melhorias
5. Valide resultado com quality gates
