# Checklist de Lançamento de Campanha Esotérica

## Pré-Lançamento

### Estratégia
- [ ] Objetivo da campanha definido (awareness, conversão, retargeting)
- [ ] Público-alvo segmentado
- [ ] Orçamento alocado por plataforma
- [ ] KPIs e metas definidos
- [ ] Calendário alinhado com datas esotéricas relevantes

### Criativos
- [ ] Criativos aprovados (imagem/vídeo)
- [ ] Dimensões corretas por plataforma
- [ ] Texto overlay <20% (Meta)
- [ ] Variações A/B preparadas (min 2)
- [ ] Copy de anúncio revisada

### Landing Page
- [ ] LP publicada e funcionando
- [ ] LP-conversion-checklist validado
- [ ] Links do anúncio apontando para URL correta
- [ ] UTMs configuradas (utm_source, utm_medium, utm_campaign)
- [ ] Pixel/conversão de teste disparado

### Tracking
- [ ] Meta Pixel ativo e verificado
- [ ] Google Analytics configurado
- [ ] Eventos de conversão testados
- [ ] UTMs em todos os links
- [ ] Relatório de acompanhamento preparado

## Lançamento
- [ ] Campanha publicada na plataforma
- [ ] Status: ativo e entregando impressões
- [ ] Verificar se anúncio foi aprovado (sem rejeição)
- [ ] Primeiro check em 4 horas (métricas iniciais)

## Pós-Lançamento (24h)
- [ ] Impressões sendo entregues
- [ ] CTR dentro do esperado (>1.5% Meta)
- [ ] CPC dentro do orçamento
- [ ] LP recebendo tráfego (verificar analytics)
- [ ] Nenhuma rejeição de anúncio

## Pós-Lançamento (72h)
- [ ] Análise inicial de métricas
- [ ] Identificar criativo vencedor (se A/B)
- [ ] Pausar criativos com performance ruim
- [ ] Ajustar orçamento se necessário
- [ ] Documentar aprendizados

## Compliance Esotérico
- [ ] Nenhuma claim de "cura" ou resultado garantido
- [ ] Linguagem dentro das políticas da plataforma
- [ ] Sem promessas de resultados espirituais absolutos
- [ ] Disclaimer quando necessário
