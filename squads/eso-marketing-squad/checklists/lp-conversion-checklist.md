# Checklist de Conversão para Landing Pages Esotéricas

## Above the Fold
- [ ] Proposta de valor clara em <5 segundos
- [ ] CTA primário visível sem scroll
- [ ] Headline magnética com gatilho esotérico
- [ ] Imagem hero relevante e de qualidade
- [ ] Tempo de carregamento <3 segundos

## CTAs
- [ ] Contraste do botão >4.5:1 com background
- [ ] Texto orientado a benefício (não "Enviar" ou "Clique aqui")
- [ ] CTA repetido a cada 2-3 seções
- [ ] Botão grande o suficiente para clique mobile (min 44x44px)
- [ ] Micro-copy sob o botão reduzindo objeção

## Prova Social
- [ ] Depoimentos com foto e nome real
- [ ] Avaliações/ratings visíveis
- [ ] Contadores (clientes, vendas, avaliações)
- [ ] Logos/selos de confiança
- [ ] Garantia clara e visível perto do CTA

## Urgência/Escassez
- [ ] Elemento de urgência presente (timer, estoque, data)
- [ ] Urgência contextualizada ao esotérico (lua cheia, sabbat)
- [ ] Não parecer falso/manipulativo

## Mobile
- [ ] Layout responsivo sem quebras
- [ ] Fontes legíveis (min 16px body)
- [ ] Botões acessíveis com polegar
- [ ] Imagens não cortam conteúdo importante
- [ ] Formulários fáceis de preencher

## Estética Esotérica
- [ ] Paleta coerente com arquétipo da marca
- [ ] Tipografia adequada (não Comic Sans em marca mística)
- [ ] Elementos visuais místicos com propósito (não decorativos)
- [ ] Espaço em branco criando respiro sagrado
- [ ] Consistência visual entre seções

## Técnico
- [ ] Meta pixel instalado
- [ ] Evento de conversão configurado
- [ ] Open Graph tags definidas
- [ ] Favicon
- [ ] SSL ativo (https)
