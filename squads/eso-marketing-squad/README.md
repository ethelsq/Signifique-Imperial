# eso-marketing-squad

Squad de Marketing Digital e Design de Alta Conversão para produtos esotéricos.

## Agents

| Agent | Role | Commands |
|-------|------|----------|
| **MenteSaEso01** (`@mente-sa-eso01`) | Estrategista de Marketing Esotérico | 8 commands |
| **EsoDesigner01** (`@eso-designer01`) | Designer UX/UI de Alta Conversão | 5 commands |

## Commands

### MenteSaEso01
| Command | Description |
|---------|-------------|
| `*pesquisa-mercado` | Pesquisa completa: concorrência, tendências, sizing, público |
| `*estrategia-trafego` | Planejar estrutura de campanhas Meta/Google/TikTok |
| `*criar-lp` | Recomendar tipo de LP, estrutura, copy |
| `*analisar-campanha` | Diagnosticar performance com recomendações |
| `*posicionar-marca` | Definir arquétipo, storytelling, diferenciação |
| `*planejar-funil` | Desenhar funil completo TOFU/MOFU/BOFU |
| `*criar-copy` | Gerar copy para anúncios, LPs, emails |
| `*calendario-esoterico` | Mapear sazonalidade e oportunidades |

### EsoDesigner01
| Command | Description |
|---------|-------------|
| `*design-lp` | Wireframe completo de LP com especificações |
| `*design-criativo` | Briefing de criativos para ads |
| `*audit-ux` | Auditoria UX com checklist de conversão |
| `*design-email` | Estrutura visual de email marketing |
| `*paleta-esoterica` | Paleta, tipografia e identidade visual |

## Workflow Recomendado

```
1. *posicionar-marca → Define identidade da marca
2. *paleta-esoterica → Cria linguagem visual
3. *pesquisa-mercado → Entende o mercado
4. *planejar-funil → Desenha jornada do cliente
5. *calendario-esoterico → Mapeia sazonalidade
6. *estrategia-trafego → Estrutura campanhas
7. *criar-lp + *design-lp → Cria landing pages
8. *criar-copy + *design-criativo → Cria anúncios
9. *analisar-campanha → Otimiza resultados
```
